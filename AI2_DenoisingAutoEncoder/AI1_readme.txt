3주차 수업에 사용할 AI실습 자료 및 소스코드입니다.
코드와 자료 모두 실습 전에 미리 읽어보고 오기 바랍니다.

Windows사용자는 C:\MinGW\bin 폴더 내 mingw32-make.exe를 make.exe라는 이름으로 복사하고 오세요.

예제 파일은 압축 해제 후 다음과 같이 실행할 수 있습니다.

./NeuralDigitRecognizer_solution.exe		// Windows
./NeuralDigitRecognizer_solution_mac		// Mac



소스 컴파일 방법은 다음과 같습니다.

make 				// on Windows

./NeuralDigitRecognizer.exe



make ARCH=MAC		// on Mac

./NeuralDigitRecognizer



make ARCH=LINUX		// on Linux

./NeuralDigitRecognizer



축복합니다.
